# SEO и Аналитика - Руководство

## Google Analytics

### Установлено:
- Google Analytics 4 (GA4)
- ID: `G-1PKV4BWHCS`
- Автоматическое отслеживание страниц

### Где смотреть статистику:
1. Откройте https://analytics.google.com/
2. Войдите с аккаунтом, на который зарегистрирован счетчик
3. Выберите свойство "KMFF"

### Что отслеживается:
- Просмотры страниц
- Количество посетителей
- Источники трафика
- Поведение пользователей
- Конверсии

## Яндекс Метрика

### Как добавить:
1. Создайте счетчик на https://metrika.yandex.ru/
2. Скопируйте ID счетчика (число)
3. Добавьте в `.env.local`:
   ```
   NEXT_PUBLIC_YM_ID=ваш_id
   ```
4. Пересоберите проект: `npm run build`
5. Перезапустите на сервере

### Что отслеживается:
- Тепловая карта кликов
- Вебвизор (записи сеансов)
- Отслеживание ссылок
- Точность отказов
- Отслеживание хэшей

## SEO Оптимизация

### Что добавлено:

#### 1. Meta теги
- Title с шаблоном
- Description с ключевыми словами
- Keywords для поисковиков
- Open Graph для соцсетей (Facebook, LinkedIn)
- Twitter Cards
- Robots meta

#### 2. Структурированные данные
- Schema.org Organization
- Добавлено в layout.tsx
- Помогает поисковикам понять структуру сайта

#### 3. Robots.txt
- Файл: `/public/robots.txt`
- Разрешает индексацию всех страниц
- Указывает на sitemap.xml

#### 4. Sitemap.xml
- Автоматически генерируется Next.js
- Доступен по адресу: https://kmff.kz/sitemap.xml
- Обновляется автоматически

### Регистрация в поисковиках:

#### Google Search Console
1. Откройте https://search.google.com/search-console
2. Добавьте сайт: https://kmff.kz
3. Выберите метод верификации "HTML тег"
4. Скопируйте код верификации
5. Добавьте в `layout.tsx` в поле `verification.google`
6. Пересоберите и задеплойте
7. Нажмите "Подтвердить" в Search Console
8. Отправьте sitemap: https://kmff.kz/sitemap.xml

#### Яндекс Вебмастер
1. Откройте https://webmaster.yandex.ru/
2. Добавьте сайт: https://kmff.kz
3. Выберите метод верификации "Meta-тег"
4. Скопируйте код верификации
5. Добавьте в `layout.tsx` в поле `verification.yandex`
6. Пересоберите и задеплойте
7. Нажмите "Проверить" в Вебмастере
8. Добавьте sitemap: https://kmff.kz/sitemap.xml

## Рекомендации

1. **Регулярно проверяйте аналитику** (раз в неделю)
2. **Отправляйте sitemap в поисковики** после обновлений
3. **Следите за ошибками** в Search Console / Вебмастере
4. **Добавляйте новый контент** регулярно
5. **Оптимизируйте изображения** (используйте WebP)
6. **Улучшайте скорость загрузки** страниц
7. **Используйте HTTPS** (уже настроено)
8. **Добавляйте alt теги** к изображениям
9. **Создавайте уникальные заголовки** для каждой страницы
10. **Пишите качественные описания** для страниц
